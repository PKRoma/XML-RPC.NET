XML-RPC.NET - Status

This page lists the status of bugs and enhancements.

Issues

  • Autodocumentation should report use of invalid types. For example, if an invalid type is used for the return type of an XML-RPC method, autodocumentation currently displays nothing.

Wishlist

  • Support for mapping .NET class types to and from XML-RPC struct.
  • Suuport for <nil/> extension. See OntoSys page.
  • Provide access to response headers.
  • gzip HTTP compression.
  • Determine if and how component support should be improved.
